# Mi Web Personal

Este es mi sitio web personal donde comparto pensamientos, artículos y proyectos. El sitio está construido con HTML y CSS puro, manteniendo un diseño minimalista y eficiente.

## Características

- Diseño responsive
- Blog personal
- Sección de portfolio
- Optimizado para GitHub Pages

## Estructura del Sitio

```
.
├── index.html
├── css/
│   └── style.css
├── blog.html
└── portfolio.html
```

## Desarrollo Local

Para desarrollar localmente, simplemente clona el repositorio y abre `index.html` en tu navegador:

```bash
git clone https://github.com/Chugeno/miweb.git
cd miweb
```
